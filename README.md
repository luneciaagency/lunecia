# 🌙 Lunecia

Official website for **Lunecia**, an AI automation and digital solutions agency focused on helping businesses grow through modern technology.

## 🚀 Services

- AI Automation
- AI Chatbots
- Website Development
- Search Engine Optimization (SEO)
- Business Automation
- Custom Web Applications

## 🛠️ Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Resend
- Vercel

## 📂 Features

- Modern responsive UI
- SEO optimized
- Fast performance
- Contact form with Resend
- Portfolio section
- Accessibility improvements
- Open Graph & Twitter Cards
- Sitemap & Robots.txt

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/luneciaagency/lunecia.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
